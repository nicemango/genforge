# Genforge 品牌方案

> 最后更新：2026-03-25

---

## 品牌名

**Genforge**

> 词源：Gen（生成）+ Forge（锻造）—— 用 AI 批量锻造内容

---

## 一句话定位

> **Forge content at scale with AI.**
> 用 AI 批量"锻造"内容

**中文定义**：一个自动化内容工厂，帮你批量生产 + 优化 + 分发内容，将创意转化为可持续的流量引擎。

---

## 产品定位

> Genforge is an AI-powered content factory that automates creation, optimization, and distribution — turning ideas into scalable traffic.

核心能力拆解：

- **自动生成**：AI 写作 / 多平台内容适配
- **批量生产**：像流水线一样持续输出
- **智能分发**：多渠道一键发布
- **增长优化**：爆文标题、结构、热点追踪

---

## 核心卖点

| 功能 | 说明 |
|------|------|
| **Automated Creation** | 从选题到成品内容，全流程自动化 |
| **Batch Production** | 一次生成 10 / 100 / 1000 条内容 |
| **Multi-Platform Publishing** | 小红书 / 微信公众号 / Twitter / Blog |
| **Growth Optimization** | 自动优化标题、结构、热点匹配 |
| **Always-On Engine** | 7x24 持续生产，无需人工盯守 |

## 视觉风格

### 风格关键词

- **工业感**：Forge / Factory — 锻造、批量、工业化
- **AI 科技感**：暗色背景 + 渐变，线性 / Vercel 质感
- **极简**：留白、克制、高信息密度

### 配色体系

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色（深空黑） | `#0a0a0f` | 背景、容器 |
| 次色（暗灰） | `#18181b` | 卡片、面板 |
| 强调色（烈焰紫） | `#7c2bee` | 主按钮、图标高亮 |
| 点缀色（熔岩橙） | `#f97316` | 动态指示、火焰元素 |
| 成功色 | `#16a34a` | 状态、完成 |
| 文字主色 | `#f4f4f5` | 正文 |
| 文字次色 | `#71717a` | 辅助说明 |

### 字体

- 标题：**Nunito**（圆润有力）
- 正文：**Quicksand**（现代几何感）
- 等宽：**JetBrains Mono**（代码、数据展示）

## Logo 设计方向

可交由设计师的几个思路：

1. **火焰 + F** — Forge 的锻造感，火焰代表 AI 生产能力
2. **齿轮 + AI 流线** — 工业化流水线，AI 智能调度
3. **模块拼接** — 内容块像积木一样批量产出
4. **循环流** — 内容从输入到分发的完整闭环

> 推荐方向：**火焰 + 抽象 F**，简洁有力，AI 感与工业感兼顾

---

## 产品战略定位

### 核心认知

> 不是写作工具，而是 **Content Infrastructure（内容基础设施）**

对标方向：
- ❌ 不要对标：写作助手（Jasper、Copy.ai）
- ✅ 对标：增长引擎 / 自动化系统（对标 Linear、Vercel 的产品感）

### 核心价值主张

| 旧表达 | 新表达 |
|--------|--------|
| AI 帮你写内容 | 持续生产 1000 篇内容的能力 |
| 一个写作工具 | 24/7运转的内容工厂 |
| 提高写作效率 | 降低流量获取成本 |

## 流水线（Pipeline）节点命名映射

| Pipeline 节点 | 品牌语言 | 说明 |
|--------------|---------|------|
| TREND_CRAWL | `Ignite` | 点燃：从 RSS 源采集热点火花 |
| TOPIC_SELECT | `Refine` | 提炼：AI 筛选最佳话题 |
| RESEARCH | `Forge` | 锻造：深度研究，锻造内容骨架 |
| WRITE | `Cast` | 浇铸：生成完整文章 |
| GENERATE_IMAGES | `Mold` | 塑形：配图装帧 |
| REVIEW | `Temper` | 淬炼：质量把关，强度测试 |
| PUBLISH | `Ship` | 出厂：分发至各渠道 |
| FULL_PIPELINE | `Full Forge` | 完整锻造：一键批量生产 |
